# Blog Customizer

Одностраничное веб-приложение для настройки оформления статьи с помощью панели параметров. Изменения применяются только после подтверждения пользователем.

## Демо
https://blog-customizer-project.netlify.app/

## Стек технологий
React, TypeScript, HTML5, SCSS, Webpack

## Функциональность

- Сайдбар с настройками оформления статьи
- Открытие и закрытие сайдбара по кнопке или клику вне его области
- Изменение параметров оформления без их мгновенного применения
- Применение выбранных настроек после нажатия кнопки «Применить»
- Сброс настроек к значениям по умолчанию
- Сохранение выбранных настроек при повторном открытии сайдбара
- Изменение оформления статьи с использованием CSS-переменных

## Запуск проекта

### Установка и запуск

```bash
git clone https://github.com/olesia-opiakina/blog-customizer.git
cd blog-customizer
npm install
npm start
```

### Production-сборка

```bash
npm run build
```

После выполнения команды собранная версия проекта будет находиться в папке `dist`
